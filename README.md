这是游戏《逃离鸭科夫》的一款Mod。This is a Mod of the game Escape from Duckov.

显示物品的数量和具体路径，包括：

1. 角色背包内的
2. 仓库内的
3. 角色已装备的
4. 宠物背包内的
5. 安装在武器上的（配件）
6. 弹夹内的（子弹）
7. 针包、钥匙包内的（针、钥匙）
8. 其他便携容器

希望能帮助大家整理物品！按Shift键收起或展开详情。

Displays the quantity and exact location of a specific item across all possible places, including:

1. Held in the backpack
2. Stored in the warehouse
3. Equipped on the character
4. Inside the pet backpack
5. Attached/installed on weapons
6. Inside magazines/ammo clips
7. In the injection/key case
8. In portable containers/bags

Never lose track of your items again! Press Shift to hide or show details.
